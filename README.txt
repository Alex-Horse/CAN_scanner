# CAN Scanner & CAN Generator

STM32-based CAN bus analyzer and OBD-II diagnostic tool.

## Features

- CAN bus scanner
- CAN packet generator
- OBD-II diagnostics
- Live CAN traffic monitor
- DTC reading
- OLED menu
- Rotary encoder interface

## Hardware

- STM32F103
- MCP2515 / SN65HVD230
- SSD1306 OLED
- Rotary encoder

## Project structure

Receiver/
Transmitter/

## Planned features

- PID monitor
- Freeze Frame
- CAN logging
- SD card support
